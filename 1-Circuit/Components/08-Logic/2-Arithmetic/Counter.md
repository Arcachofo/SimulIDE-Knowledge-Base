Binary counter.
Sets the output High after a number of clock pulses.

---

## Properties:
Property: (default value)

### Main:
- **Use Set Pin:** (no)
   Shows/hides "Set" pin.
<br>
- **Clock Inverted:** (no)
   Invert Clock pin.
<br>
- **Set/Reset Inverted:** (yes)
   Invert Set and Reset pins.
<br>
- **Count to:** (1)
   Number of Clock pulses to set output High.
