## Description:

Multiplexer.

---

## Properties:

Property: (default value)

### Main:
- **Address Size:** (3 Bits)
   Number of select pins.
   Determines number of channels: n = 2^Address_Size
<br>
- **Invert Inputs:** (no)
   Invert input pins.
