## Description:

Configurable header connector.

---

## Properties:

Property: (default value)

- **Id:** ()
   Identifier, all Tunnels with this Id will be connected togeter.
<br>
- **Rotated:** (false) ( Only 1.0.0 )
   Yes: pin to the right.
   No: pin to the left.
<br>
- **Bus:** (false)
   This tunnel connects to a Bus.
